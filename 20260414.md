- 관리자 계정 조회
```
docker exec -it gitlab gitlab-rails runner "['root','admin'].each do |n| u=User.find_by_username(n); puts(%Q{#{n}: #{u ? \"exists, admin=#{u.admin?}, state=#{u.state}\" : \"not-found\"}}); end"
```
